📌 Emplora – Employee Management System
🚀 Overview
Emplora is a modern Employee Management System designed to streamline HR operations such as employee management, attendance tracking, leave handling, and payroll generation. It supports both Admin and Employee roles with dedicated dashboards.
________________________________________
✨ Features
🔐 Authentication & Authorization
•	Role-based login (Admin / Employee) 
•	Secure authentication using tokens 
•	Protected routes with React Router 
👨‍💼 Employee Management (Admin)
•	Add, edit, delete employees 
•	Department-based filtering 
•	Employee status management (Active/Inactive) 
📅 Attendance Management
•	Daily check-in / check-out system 
•	Attendance history tracking 
•	Monthly attendance stats 
🌴 Leave Management
•	Apply for leave (Employee) 
•	Approve / reject leave (Admin) 
•	Leave type tracking (Sick, Casual, Annual) 
💰 Payroll System
•	Generate payslips (Admin) 
•	View payslip history (Employee) 
•	Printable payslip feature 
⚙️ Profile & Settings
•	Update profile information 
•	Change password functionality 
________________________________________
